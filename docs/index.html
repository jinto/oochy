핵심 기능

왜 KittyPaw인가요?

n8n, Zapier 같은 자동화 도구의 데스크톱 네이티브 대안.
AI가 코드를 쓰고, 샌드박스에서 안전하게 실행합니다.

🎯

코드 없이 자동화

스킬 갤러리에서 원하는 자동화를 설치하고, 텔레그램 토큰만 넣으면 바로 동작합니다.

🧠

학습하는 AI

반복 패턴을 감지해 자동화를 제안하고, 매주 내 관심사를 분석합니다. 쓸수록 나를 알아가는 개인 비서.

🔒

안전한 실행

모든 스킬은 QuickJS 샌드박스 + OS 커널 격리 (Seatbelt/Landlock) 안에서 실행됩니다.

🔗

스킬 체이닝

여러 스킬을 순차 파이프라인으로 연결. 데이터 수집 → 요약 → 발송을 한 번에.

🌐

웹 검색

스킬이 웹을 검색하고, 페이지를 읽고, AI로 요약합니다. 팩트체크도 자동으로.

🏠

로컬 LLM 지원

Ollama, LM Studio 등 내 컴퓨터의 로컬 모델을 연결하세요. API 비용 0원. 데이터가 내 컴퓨터를 떠나지 않습니다.

학습하는 AI

KittyPaw는 기억합니다

쓸수록 내 습관을 이해하고, 내가 요청하기 전에 먼저 제안합니다

🔄

반복 패턴 감지

같은 종류의 요청을 반복하면 AI가 패턴을 감지하고 자동화를 제안합니다.

환율 알려줘 달러 가격이 얼마야? 오늘 환율 어때? 💡 환율을 자주 확인하시네요.
매일 아침 자동으로 알려드릴까요?
📊

주간 관심사 리포트

매주 일요일, 이번 주 내가 무엇에 관심을 가졌는지 자동으로 정리해 드립니다.

사용 방법

3단계로 끝납니다

설치하고 설정하면 고양이가 알아서 일합니다

📦
1

스킬 설치

갤러리에서 원하는 스킬을 선택해 클릭 한 번으로 설치합니다.

⚙️
2

설정 입력

텔레그램 토큰, 스케줄, API 키만 채웁니다. 코드는 건드리지 않아도 됩니다.

✨
3

자동 실행

설정한 일정에 따라 자동으로 실행되고 결과를 알림으로 받습니다.

활용 예시

이렇게 쓸 수 있습니다

설치하고 설정만 채우면 바로 동작하는 자동화들

1

매일 아침 시장 리포트

ETF 데이터를 수집하고 AI가 요약해서 텔레그램으로 보내줍니다. 평일 오전 8시 자동 발송.

2

RSS 뉴스 요약

Hacker News, TechCrunch 등 RSS 피드의 새 글을 AI가 요약. 이미 본 글은 자동 필터링.

3

서비스 다운 알림

내 웹사이트/API를 5분마다 체크. 다운되면 즉시 텔레그램 알림. 복구되면 다시 알림.

4

날씨 브리핑

매일 아침 7일 예보를 자연어로 알려줍니다. API 키 불필요 (Open-Meteo 무료 API).

5

나만의 자동화 만들기

kittypaw teach "매일 로또 당첨번호 확인해서 알려줘" — AI가 코드를 짜고 테스트하고 등록합니다.
