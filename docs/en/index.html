Core Features

Why KittyPaw?

A desktop-native alternative to automation tools like n8n and Zapier.
AI writes the code, runs it safely in a sandbox.

🎯

Automate Without Code

Install any automation from the skill gallery, enter your Telegram token, and it works immediately.

💬

AI Chat

Chat with Claude and OpenAI right from your desktop. Ask questions, analyze data, and direct your automations.

🔒

Safe Execution

Every skill runs inside a QuickJS sandbox with OS kernel isolation (Seatbelt/Landlock).

🔗

Skill Chaining

Connect multiple skills into sequential pipelines. Collect data → summarize → deliver, all in one flow.

🌐

Web Search

Skills can search the web, read pages, and summarize with AI. Fact-checking included, automatically.

🏠

Local LLM Support

Connect local models via Ollama, LM Studio, and more. Zero API cost. Your data never leaves your machine.

Use Cases

What you can do with it

Ready-to-run automations — install, configure, go

1

Daily Morning Market Report

Fetches ETF data, summarizes it with AI, and sends it to Telegram. Auto-delivered weekdays at 8 AM.

2

RSS News Digest

AI summarizes new posts from Hacker News, TechCrunch, and other RSS feeds. Already-read articles are filtered automatically.

3

Service Downtime Alerts

Checks your website or API every 5 minutes. Instant Telegram alert when it goes down, another when it recovers.

4

Weather Briefing

Delivers a 7-day forecast in plain language every morning. No API key required (uses the free Open-Meteo API).

5

Build Your Own Automation

kittypaw teach "check lottery numbers daily and notify me" — AI writes the code, tests it, and registers it.
